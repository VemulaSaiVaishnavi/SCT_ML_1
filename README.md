# House Price Prediction using Linear Regression

## Overview
This project implements a Multiple Linear Regression model to predict house prices based on:
- Area (Square Footage)
- Number of Bedrooms
- Number of Bathrooms

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Machine Learning Algorithm
- Linear Regression

## Dataset Features
- Area
- Bedrooms
- Bathrooms
- Price

## Project Workflow
1. Data Loading
2. Data Preprocessing
3. Train-Test Split
4. Model Training
5. Prediction
6. Model Evaluation
7. Data Visualization

## Evaluation Metrics
- Mean Squared Error (MSE)
- R² Score
